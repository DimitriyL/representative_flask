Introduction to Neo4j

Popular graph database management system
Uses the Cypher query language to add, retrieve, and remove data points
Implemented using Java
Meets the standards of ACID (Atomicity, Consistency, Isolation, Durability)

Getting Started

1. Install and download off of neo4j.com
2. Launch the program and create a new project
3. Click “Open Browser” to open a new window to toggle with the database
4. Proceed, either with tutorials or by writing original Cypher code
